# Publishing extensions

## Publishing Commands

`vsce login <publisher>`
<!-- Login to Publisher -->
`vsce package`
<!-- myExtension.vsix generated -->
`vsce publish major | minor | patch`
<!-- <publisherID>.myExtension published to VS Code MarketPlace -->
